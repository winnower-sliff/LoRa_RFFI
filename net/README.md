# Net 神经网络模型

该文件夹包含了项目中使用到的各种神经网络模型实现。

## 文件说明

- [TripletNet.py](TripletNet.py)：三元组网络模型，用于学习嵌入表示和相似性度量
- [net_DRSN.py](net_DRSN.py)：DRSN网络模型实现
- [net_MobileNet.py](net_MobileNet.py)：MobileNet模型实现，专为移动和嵌入式设备设计的轻量级网络
- [net_prune.py](net_prune.py)：专为剪枝优化设计的网络模型
- [net_resnet.py](net_resnet.py)：ResNet网络模型实现，采用残差连接解决深度网络训练问题
- [net_wt.py](net_wt.py)：WT网络模型实现

这些模型文件实现了项目中使用的各种深度学习架构，为不同的任务和需求提供支持。
